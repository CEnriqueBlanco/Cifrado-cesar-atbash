NEX-01 (inicio de app)
- Importa Flask, render_template y request desde flask.
- Crea la instancia principal de la aplicación: app = Flask(__name__).
- Define PRINTABLE_ASCII con los caracteres ASCII imprimibles (32 a 126).
- Efecto práctico: deja un alfabeto por defecto listo para usar en cifrado/descifrado.

NEX-02 (normalización de charset)
- Recibe un string (charset) con los caracteres elegidos por el usuario.
- Inicializa seen (set) para registrar caracteres ya vistos.
- Inicializa unique (lista) para conservar el orden de aparición.
- Recorre carácter por carácter:
  - Si no estaba en seen, lo agrega a seen y a unique.
  - Si ya estaba, lo ignora.
- Devuelve ''.join(unique).
- Efecto práctico: evita duplicados en el alfabeto sin perder orden.

NEX-03 (algoritmo César)
- Firma: caesar_cipher(text, charset, shift, decrypt=False).
- Si charset está vacío, devuelve text sin cambios.
- size = len(charset): tamaño del alfabeto activo.
- shift = shift % size: normaliza desplazamientos grandes/negativos.
- Si decrypt=True, invierte el desplazamiento con shift = -shift.
- Recorre el texto:
  - Si el carácter está en charset:
    - Obtiene índice original con charset.index(char).
    - Calcula nuevo índice circular: (index + shift) % size.
    - Agrega el carácter transformado.
  - Si no está en charset, lo conserva igual.
- Devuelve ''.join(result).
- Efecto práctico: cifra o descifra según el desplazamiento y operación.

NEX-04 (algoritmo Atbash)
- Firma: atbash_cipher(text, charset).
- Si charset está vacío, devuelve text sin cambios.
- size = len(charset).
- Recorre el texto:
  - Si el carácter está en charset:
    - Obtiene índice normal.
    - Calcula índice espejo: size - 1 - index.
    - Agrega el carácter espejo.
  - Si no está en charset, lo conserva.
- Devuelve ''.join(result).
- Efecto práctico: aplica sustitución simétrica (cifrar y descifrar son iguales).

NEX-05 (ruta principal /)
- Define endpoint '/' con métodos GET y POST.
- Inicializa valores por defecto:
  - module='caesar', operation='encrypt', shift=3,
  - charset=PRINTABLE_ASCII, text/output/error/info vacíos.
- Si request.method == 'POST':
  - Lee campos del formulario: module, operation, text, charset.
  - Normaliza charset con normalize_charset.
  - Si hubo duplicados removidos, coloca mensaje en info.
  - Si charset queda vacío, coloca error y no procesa cifrado.
  - Si módulo es 'caesar':
    - Lee shift, limpia espacios y convierte a int.
    - Si falla conversión, registra error de llave inválida.
    - Si es válido, llama caesar_cipher con decrypt según operation.
  - Si módulo es 'atbash':
    - Llama atbash_cipher.
  - Si módulo no coincide, marca error de módulo no válido.
- Retorna render_template('index.html', ...)
  enviando todas las variables necesarias a la plantilla.
- Efecto práctico: controla flujo completo de la página y lógica del formulario.
